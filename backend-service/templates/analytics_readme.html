Traffic Summary KPIs

Top-line activity and short-term momentum for the portfolio. These KPIs give a quick read on how much traffic the site is getting right now and whether recent interest is accelerating or cooling off.

Total Visitors (All Time)
Total number of recorded visit events across all time. This is the master counter used for the dashboard headline.
Today's Visitors
Total visit events recorded for the current calendar day in the configured timezone.
Visitors Since 7 Days
Rolling 7-day sum of visit events including today. Useful for week-over-week momentum.
Visitors Since 30 Days
Rolling 30-day sum of visit events including today. Captures monthly demand and seasonality.
Repeat Visitors (Last 24 Hours)
Distinct IPs that have a repeat-visit flag within the last 24 hours. This highlights return behavior in the most recent day.

Highest Performing Segments

Highlights from the strongest day and the best-performing sources. These are directional indicators for where attention is concentrated.

Peak Visit Day
Date with the highest visit count in the last 30 days. Helpful for identifying spikes tied to content releases or promotions.
Top Country
Country with the most recorded visits over all time. Useful for regional targeting and localization decisions.
Top Source
Most frequent traffic source or referrer. Helps validate which channels are driving traffic.

Performance Over Time

Rolling averages that capture stability and repeat behavior. These smooth short-term spikes and give a more reliable baseline for planning.

Avg. Visits / Day
Average daily visit events across the last 30 days. Good for monitoring daily demand.
Avg. Visits / Week
Average weekly visit events across the last 12 months. Useful for seasonality and long-term trend comparisons.
Avg. Repeat Visitors / Day
Average daily repeat IPs across the last 6 months. Indicates how sticky the audience is.

Visualizations

Charts that show trends, distribution, and top segments. Use these to visualize shifts in traffic volume and where visitors are coming from.

Top 5 Locations List
Top cities and regions by visit count, displayed as a ranked list for quick scanning.
Visitor Trends
Time-series trends for 7D, 1M, 1Y, and 5Y views. Use this to compare short-term spikes to long-term growth.
Visitors by Country
Top 10 countries by visit volume. Highlights geographic concentration of traffic.
Visitors by Source
Top 5 sources or referrers by visit volume. Use this to measure channel effectiveness.

Data Notes

The analytics database stores one row per visit event. Repeat-visit metrics are derived from the visit history of the same IP within recent windows, and time windows are evaluated using the configured timezone.

Time Windows
7-day and 30-day metrics are rolling windows that include today. The last 24 hours is a moving window relative to the current time.
Repeat Visitors
An IP is marked as repeat when it has a prior visit within the last 24 hours at the moment the event is recorded.
